| Техника | Правило | Формула |
|:---|:---|:---|
| **Эквивалентное разделение** | Разбить входные данные на классы, где поведение одинаково. Выбрать **один представитель** из каждого класса. | N = количество классов эквивалентности |
| **Граничные значения** | Проверить значения на границах диапазонов и сразу за ними. | L-1, L, L+1, R-1, R, R+1 |

**Правила верификации таблицы:**
1. **Полнота:** Каждая комбинация условий (2^n правил) должна быть рассмотрена или явно исключена.
2. **Непротиворечивость:** Для одной комбинации условий не может быть двух разных наборов действий.
3. **Минимизация:** Склеить правила, где результат не зависит от какого-то условия (обозначить «любое»).

function calculateDelivery($orderTotal, $city, $distance = 0) {
    // Проверка минимальной и максимальной суммы
    if ($orderTotal < 500) {
        return ['error' => 'Минимальная сумма заказа 500 рублей'];
    }
    if ($orderTotal > 100000) {
        return ['error' => 'Максимальная сумма заказа 100 000 рублей'];
    }
    
    // Расчёт доставки по городу
    if ($city == 'city') {
        if ($orderTotal >= 5000) {
            $deliveryCost = 0; // Бесплатно
        } else {
            $deliveryCost = 300; // Фиксированная
        }
    }
    
    // Расчёт доставки за город
    if ($city == 'suburb') {
        if ($distance > 50) {
            return ['error' => 'Доставка за город возможна только до 50 км'];
        }
        
        $deliveryCost = 500 + ($distance * 30);
        
        // Бесплатная доставка за город только при заказе от 10 000
        if ($orderTotal >= 10000) {
            $deliveryCost = 0;
        }
    }
    
    return ['delivery_cost' => $deliveryCost];
}

### Тест-кейс TC-COMBINED-02

**Название:** Обычный клиент, сумма на границе бесплатной доставки (5000) → скидки нет

**Техники:** ГЗ (5000) + Таблица решений (правило 4: нет скидок)

**Входные данные:**
| Параметр | Значение |
|:---|:---|
| orderTotal | 5000 |
| loyalty | false |
| city | city |

**Ожидаемый результат:**
- Скидка на доставку = 0%
- Стоимость доставки = 0 (бесплатно, т.к. orderTotal >= 5000 для города)

**Примечание:** Это граничный случай — 5000 даёт бесплатную доставку, но не даёт скидки (скидка только >10000).
