## Теоретическая справка

### Техники эквивалентного разделения (ЭП) и граничных значений (ГЗ)

| Техника | Правило | Формула |
|:---|:---|:---|
| **Эквивалентное разделение** | Разбить входные данные на классы, где поведение одинаковое. Выбрать **один представитель** из каждого класса. | N = количество классов эквивалентности |
| **Граничные значения** | Проверить значения на границах диапазонов и сразу за ними. | L-1, L, L+1, R-1, R, R+1 |

### Таблица решений (Decision Table)

**Структура:**
- Условия (входные параметры в булевом виде)
- Действия (результаты)
- Правила (комбинации условий)

# delivery_calculator.py

from dataclasses import dataclass
from typing import Tuple, Optional


@dataclass
class DeliveryResult:
    """Результат расчёта доставки."""
    delivery_cost: float
    discount_percent: float
    final_total: float
    message: Optional[str] = None


class DeliveryCalculator:
    """
    Калькулятор стоимости доставки и скидок.
    
    Правила:
    1. Минимальная сумма заказа: 500 рублей
    2. Максимальная сумма заказа: 100 000 рублей
    3. Скидка зависит от суммы заказа и статуса пользователя
    4. Доставка зависит от города и суммы после скидки
    """
    
    # Константы
    MIN_ORDER = 500
    MAX_ORDER = 100_000
    CITY_DELIVERY_FEE = 300
    SUBURB_DELIVERY_BASE = 500
    SUBURB_DELIVERY_PER_KM = 30
    MAX_DISTANCE = 50
    
    def calculate_delivery(
        self,
        order_total: float,
        city: str,
        distance: float = 0,
        user_status: str = 'regular',
        has_promocode: bool = False
    ) -> DeliveryResult:
        """
        Рассчитывает стоимость доставки и итоговую сумму.
        
        Args:
            order_total: Сумма заказа
            city: 'city' (город) или 'suburb' (пригород)
            distance: Расстояние от города (для пригорода)
            user_status: 'regular', 'vip', 'new'
            has_promocode: Есть ли промокод на скидку 10%
        
        Returns:
            DeliveryResult с результатами расчёта
        """
        
        # 1. Проверка минимальной и максимальной суммы
        if order_total < self.MIN_ORDER:
            return DeliveryResult(
                delivery_cost=0,
                discount_percent=0,
                final_total=order_total,
                message=f"Минимальная сумма заказа {self.MIN_ORDER} рублей"
            )
        
        if order_total > self.MAX_ORDER:
            return DeliveryResult(
                delivery_cost=0,
                discount_percent=0,
                final_total=order_total,
                message=f"Максимальная сумма заказа {self.MAX_ORDER} рублей"
            )
        
        # 2. Расчёт скидки
        discount_percent = 0
        
        if user_status == 'vip':
            discount_percent = 15
        elif user_status == 'regular':
            if order_total > 5000:
                discount_percent = 10
            else:
                discount_percent = 5
        # new users: 0% скидка
        
        # Промокод даёт дополнительные 10%, но не более 25% суммарно
        if has_promocode:
            discount_percent += 10
            if discount_percent > 25:
                discount_percent = 25
        
        discount_amount = order_total * (discount_percent / 100)
        after_discount = order_total - discount_amount
        
        # 3. Расчёт доставки
        delivery_cost = 0
        
        if city == 'city':
            # Бесплатная доставка по городу при сумме >= 5000
            if after_discount >= 5000:
                delivery_cost = 0
            else:
                delivery_cost = self.CITY_DELIVERY_FEE
                
        elif city == 'suburb':
            # Проверка максимального расстояния
            if distance > self.MAX_DISTANCE:
                return DeliveryResult(
                    delivery_cost=0,
                    discount_percent=discount_percent,
                    final_total=after_discount,
                    message=f"Доставка за город возможна только до {self.MAX_DISTANCE} км"
                )
            
            # Бесплатная доставка за город при сумме >= 10000
            if after_discount >= 10000:
                delivery_cost = 0
            else:
                delivery_cost = self.SUBURB_DELIVERY_BASE + (distance * self.SUBURB_DELIVERY_PER_KM)
        
        # 4. Итоговая сумма
        final_total = after_discount + delivery_cost
        
        return DeliveryResult(
            delivery_cost=delivery_cost,
            discount_percent=discount_percent,
            final_total=round(final_total, 2)
        )
