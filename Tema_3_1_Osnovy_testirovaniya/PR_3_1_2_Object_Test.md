## Теоретическая справка

### Что такое «объект тестирования»?

**Объект тестирования** — это конкретный компонент, модуль, функция или система в целом, которую вы проверяете.

### Три подхода к тестированию

| Подход | Знание внутреннего устройства | Кто использует | Что проверяем |
|:---|:---|:---|:---|
| **Чёрный ящик** | Нет | Тестировщик, заказчик | Функции, UI, требования |
| **Белый ящик** | Да (полный доступ к коду) | Разработчик | Логику внутри методов, ветвления |
| **Серый ящик** | Частичное (API, БД) | QA-автоматизатор | Взаимодействие модулей |

## Объект тестирования (Python)

В качестве объекта тестирования возьмём **функцию расчёта итоговой суммы заказа со скидкой** (аналог предыдущих занятий, но на Python).

### Листинг 1. Исходный код для тестирования

```python
# order_calculator.py

class OrderCalculator:
    """
    Калькулятор стоимости заказа со скидками и доставкой.
    """
    
    def calculate(self, subtotal: float, user_tier: str, delivery_fee: float = 300, use_promocode: bool = False) -> float:
        """
        Рассчитать итоговую сумму заказа с учётом скидок и доставки.
        
        Args:
            subtotal: Сумма товаров без скидки
            user_tier: Уровень пользователя ('bronze', 'silver', 'gold', 'guest')
            delivery_fee: Стоимость доставки (по умолчанию 300)
            use_promocode: Использовать ли промокод на 10% (только для gold)
        
        Returns:
            Итоговая сумма (округлённая до 2 знаков)
        """
        # 1. Рассчитываем скидку в зависимости от уровня
        discount_percent = 0
        
        if user_tier == 'gold':
            discount_percent = 15
        elif user_tier == 'silver':
            discount_percent = 10
        elif user_tier == 'bronze':
            discount_percent = 5
        else:
            discount_percent = 0
        
        # 2. Дополнительная скидка по промокоду (только для gold)
        if use_promocode and user_tier == 'gold':
            discount_percent += 10
        
        # 3. Применяем скидку
        discount = subtotal * (discount_percent / 100)
        after_discount = subtotal - discount
        
        # 4. Бесплатная доставка для сумм больше 5000
        if after_discount > 5000:
            delivery_fee = 0
        
        # 5. Итог
        return round(after_discount + delivery_fee, 2)

import asyncio
import aiohttp
import pytest


class TestAsyncApi:
    """Асинхронные тесты API (серый ящик + нагрузка)."""
    
    async def fetch_calculate(self, session, data):
        url = "http://localhost:8000/api/calculate"
        async with session.post(url, json=data) as response:
            return await response.json()
    
    async def test_concurrent_requests(self):
        """Проверка обработки 10 параллельных запросов."""
        async with aiohttp.ClientSession() as session:
            tasks = []
            for i in range(10):
                data = {
                    "subtotal": 10000,
                    "user_tier": "gold",
                    "use_promocode": True
                }
                tasks.append(self.fetch_calculate(session, data))
            
            results = await asyncio.gather(*tasks)
            
            for result in results:
                assert result["success"] is True
                assert result["total"] == 7500
